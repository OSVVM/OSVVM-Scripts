Support
#######

Get support for OSVVM at `osvvm.org <https://osvvm.org/>`__.

See also the following resources.


.. topic:: OSVVM Forum

   OSVVM offers a `forum <https://osvvm.org/forums/forum/os-vvm>`__ with an active community.

.. topic:: OSVVM Discussions

   For interactive discussion, see OSVVM's Gitter channel at
   `OSVVM Lobby <https://app.gitter.im/#/room/#OSVVM_Lobby:gitter.im>`__


.. topic:: Reporting Bugs

   In case of a "bug" please create a **problem report** at
   `OSVVM's Issue Tracker <https://github.com/OSVVM/OsvvmLibraries/issues>`__

.. topic:: Suggesting Enhancements

   In case of missing features or feature suggestions, please create a **feature request** at
   `OSVVM's Issue Tracker <https://github.com/OSVVM/OsvvmLibraries/issues>`__.

   OSVVM is open for suggestions via GitHub **Pull Requests**.

   .. hint::

      In case of bigger contributions or modifications to OSVVM's code base, please first start a **feature request** via
      issue tracker to discuss the proposed solution and how to integrate the intended changes best into OSVVM's
      existing code.
